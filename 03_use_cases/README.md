# Use cases

Maps to the **Use cases** section of the [cookbook website](https://dev.meta.ai/docs/getting-started/cookbook#use-cases).
End-to-end patterns: multimodal perception, orchestration, and full applications.

## Recipes

| # | Recipe | What it does |
|---|--------|--------------|
| [01](01_chart-explainer/) | Chart analysis | Read charts and extract structured data from images. |
| [02](02_screenshot_bugfix/) | Error screenshot fix | Diagnose a bug from an error screenshot and fix it. |
| [03](03_visionclaw/) | Smart glasses with OpenClaw | Hands-free look-and-ask on Ray-Ban Meta glasses. |
| [04](04_doc_generation/) | Generating slides | Generate a slide deck from a prompt or source content. |
| [05](05_web_design/) | Browser-verified web design | Build a website with a coding agent that checks its own work in a real browser. |
| [06](06_iterative_game_dev/) | Iterative game dev | Build a browser game end-to-end with a coding agent, verified in a real browser. |
| [07](07_sandbox_execution/) | Sandboxed execution | Execute model-generated code in a sandbox. |
| [08](08_multi_agent_orchestration/) | Multi-agent product studio | Orchestrate a four-profile Hermes team (PM, backend, frontend, tech writer) that coordinates through a shared Kanban board. |
| [09](09_one-shot-game_dev/) | One-shot game dev | Build a complete 3D browser game in a single pass — an AGENTS.md plus one structured prompt, no iteration loop. |
| [10](10_perception_grounding/) | Perception grounding | Identify food items in a fridge photo, pin interactive health-score dots at their pixel locations, and generate a self-contained HTML overlay. |
| [11](11_github_repo_agent/) | GitHub repo agent | Autonomous GitHub Actions bot — triage, PR review, AI-slop detection, and bug-fix PRs — built on OpenCode + Muse Spark. |
| [12](12_computer_use/) | Computer use | Drive a Linux desktop from screenshots — the agent finds an app, opens it, and plays it, clicking through a Cua sandbox. |
| [13](13_macos_cua/) | macOS computer use | Drive a real Mac from screenshots with `metacua`, a native computer-use agent (Swift + Python) that clicks, types, and works out a GUI app on its own. |

Recipes 01–05 and 07 map to live website tiles; 06 (iterative game dev), 08 (multi-agent
product studio), 09 (one-shot game dev), 10 (perception grounding), 11 (GitHub repo agent),
12 (computer use), and 13 (macOS computer use) are in-repo recipes with no live tile.
